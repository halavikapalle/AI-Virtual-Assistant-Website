# Wednesday AI Voice Assistant 🤖🎤

A browser-based AI Virtual Voice Assistant built using HTML, CSS, and JavaScript.
The assistant can recognize voice commands, respond with speech, open websites, search Google, provide weather updates, play YouTube content, and improve productivity using voice interaction.

# ⚙️ How It Works

1. User clicks the **Ask Me** button.
2. Browser microphone access is enabled.
3. Voice command is captured using Web Speech API.
4. JavaScript processes the command.
5. Assistant responds using Speech Synthesis.
6. Requested action is performed.
