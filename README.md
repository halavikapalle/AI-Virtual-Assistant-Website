# 🤖 Wednesday – AI Virtual Voice Assistant

A browser-based AI Virtual Voice Assistant built using HTML, CSS, and JavaScript.
The assistant responds to voice commands, performs web searches, opens applications, provides weather updates, and improves productivity using voice interaction.

---

## 🚀 Features

* 🎤 Voice Recognition using Web Speech API
* 🔊 Speech Response (Text-to-Speech)
* 🌦 Real-Time Weather Updates
* 🔍 Smart Google Search
* 📺 YouTube Search & Control
* 🎵 Spotify Access
* ⏰ Pomodoro Productivity Timer
* 🧮 Calculator Access
* 💻 GitHub & LinkedIn Quick Access
* 🎮 Quick Relaxation Games
* 📱 Responsive Modern UI

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript
* Web Speech API
* Fetch API

## ⚙️ How to Run the Project

### 1️⃣ Clone Repository

```bash
git clone https://github.com/halavikapalle/AI-Virtual-Assistant-Website/new/main?filename=README.md
```

### 2️⃣ Open Project Folder

```bash
cd AI-Virtual-Assistant
```

### 3️⃣ Run Project

Open `index.html` in your browser.

OR use VS Code Live Server:

* Install Live Server Extension
* Right click `index.html`
* Click **Open with Live Server**

## 🎙 Supported Voice Commands

| Command                | Action                    |
| ---------------------- | ------------------------- |
| "hello"                | Greets the user           |
| "time"                 | Tells current time        |
| "date"                 | Tells today's date        |
| "weather in Hyderabad" | Shows weather updates     |
| "open youtube"         | Opens YouTube             |
| "play songs"           | Searches songs on YouTube |
| "search for AI news"   | Google search             |
| "open spotify"         | Opens Spotify             |
| "open github"          | Opens GitHub              |
| "open linkedin"        | Opens LinkedIn            |
| "calculator"           | Opens calculator          |
| "pomodoro"             | Opens productivity timer  |
| "games"                | Opens browser games       |
